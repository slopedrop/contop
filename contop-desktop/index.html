Tailscale — Remote Access

Optional

Tailscale creates a private network between your devices so you can control your desktop from anywhere — not just the same Wi-Fi.

1

Create a Tailscale Account

Visit tailscale.com and sign up with your Google, Microsoft, or GitHub account. It's free for personal use.

2

Install on This Computer

Download and install Tailscale for your operating system. Once installed, sign in with the same account you just created.

3

Install on Your Phone

Get the Tailscale app from the App Store (iOS) or Google Play (Android). Sign in with the same account so both devices join the same network.

4

You're Connected

Both devices now share a private network. Contop will automatically detect Tailscale and use it for stable remote connections. You'll see a green "Active" badge on the Home page.

Docker — Sandboxed Execution

Optional

Docker lets Contop run potentially dangerous commands in isolated containers instead of directly on your system. This keeps your computer safe while still letting the AI do its work.

1

Install Docker Desktop

Download Docker Desktop for your operating system. It includes everything needed to run containers.

2

Automatic Start & Detection

Contop automatically starts Docker Desktop when needed — no manual launch required. Dangerous commands flagged by the safety evaluator will run in sandboxed containers with memory limits, network isolation, and read-only filesystems.

3

No Docker? No Problem

If Docker isn't installed, Contop falls back to running commands directly with restricted permissions, shorter timeouts, and path-based safety checks.

CLI Subscriptions — Use Your Existing Plan

Optional

If you have a Claude Max, Gemini Advanced, or ChatGPT Plus subscription, Contop can route AI requests through the provider's CLI instead of consuming API credits. Install the CLI, authenticate, then enable subscription mode in Settings.

1

Install a Provider CLI

Install one or more CLIs for the providers you subscribe to:

npm install -g @anthropic-ai/claude-code  # Claude Max / Pro
npm install -g @google/gemini-cli    # Gemini Advanced
npm install -g @openai/codex              # ChatGPT Plus / Pro
2

Authenticate with Your Account

Log in to each CLI so it can use your subscription:

claude /login       # opens browser to sign in
gemini auth login   # authenticates with your Google account
codex auth          # authenticates with your OpenAI account
3

Enable in Settings

Go to the Settings page and toggle the auth mode from "API Key" to "Subscription" for each provider. The proxy starts automatically with the server and routes requests through the CLI.
